What is CSS?

CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files. Internal stylesheets use the style attribute within individual HTML tags A single CSS file can contain rules for many different web pages CSS selectors determine which HTML elements will be styled by the properties and values listed in a rule set There are three types of CSS selectors:

The CSS Selectors

Class Selector

Using (.) for class selector.

This is a paragraph with class "center".

ID Selector

Using (#) for id selector .

This is a paragraph with an ID called "demo".

Attribute Selector

[attribute=value] - selects elements with the specified attribute and value.

Universal Selector

(*) - selects all elements on a page.

Grouped Selector

, - separates selectors so that you can combine several selectors into one rule.
